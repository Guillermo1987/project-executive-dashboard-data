# 📊 Proyecto 8: Dashboard Ejecutivo 360° (Business Intelligence)

## Resumen del Proyecto
Este proyecto es la base de datos y la estructura analítica para un **Dashboard Ejecutivo 360°**, diseñado para proporcionar una visión integral del rendimiento del negocio, combinando métricas de **Finanzas, Marketing y Ventas (RevOps)**. Demuestra mi capacidad para definir, calcular y estructurar los KPIs más críticos para la toma de decisiones a nivel de liderazgo.

Este proyecto es el más relevante para roles de **Business Intelligence Analyst** o **RevOps Manager** en empresas internacionales, ya que se centra en la visión estratégica y la comunicación de datos.

## Habilidades Demostradas
*   **Business Intelligence (BI):** Definición de un conjunto de KPIs interconectados que reflejan la salud del negocio.
*   **Revenue Operations (RevOps):** Integración de datos de diferentes departamentos (Marketing, Ventas, Finanzas) en una única fuente de verdad.
*   **Análisis Financiero:** Cálculo de métricas clave como Margen Bruto, CAC (Costo de Adquisición de Cliente) y Churn Rate.
*   **Preparación de Datos:** Uso de Python (Pandas) para la generación de datos sintéticos realistas y la ingeniería de características (Feature Engineering).
*   **Data Storytelling:** Estructura de datos lista para ser visualizada en Power BI, Tableau o Looker.

## Estructura del Repositorio
*   `generate_data.py`: Script de Python que genera los datos sintéticos mensuales para 12 meses.
*   `executive_dashboard_data.csv`: El *dataset* final, que contiene todas las métricas financieras y operacionales clave.
*   `dashboard_data_sample.md`: Muestra de las primeras filas del *dataset* final.
*   `README.md`: Este archivo.

## KPIs Clave Incluidos
| Categoría | Métrica | Descripción |
| :--- | :--- | :--- |
| **Finanzas** | Revenue, Profit, Gross Margin % | Rendimiento económico y eficiencia de costos. |
| **Marketing** | Marketing Spend, New Customers | Inversión y resultados de las campañas de adquisición. |
| **RevOps** | CAC (Cost of Acquisition), Churn Rate | Eficiencia de la inversión y retención de clientes. |

## Cómo Utilizar
1.  Clonar el repositorio.
2.  El archivo `executive_dashboard_data.csv` está listo para ser importado directamente a tu herramienta de BI preferida (Power BI, Tableau, etc.).
3.  Utiliza las métricas para crear visualizaciones de tendencias mensuales, comparativas de rendimiento y análisis de correlación entre Marketing Spend y Revenue.
